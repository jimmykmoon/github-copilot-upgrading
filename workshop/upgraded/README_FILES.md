# upgraded 폴더 설명

이 폴더는 레거시(legacy) Python 프로젝트를 최신 Python 환경으로 업그레이드하기 위한 작업 공간입니다.

## 주요 내용
- `legacy` 폴더의 모든 소스 코드, 문서, 테스트, 패키징 파일이 복사되어 있습니다.
- Python 2.5 기반의 구식 문법과 패키징 방식을 최신 Python 버전에 맞게 변환하는 실습을 진행합니다.
- 실제 코드(`guachi/`), 문서(`docs/`), 테스트(`tests/`), 패키징 파일(`setup.py`, `MANIFEST.in` 등)이 포함되어 있습니다.
- 업그레이드 및 마이그레이션 작업은 이 폴더에서 이루어집니다.

## 활용 예시
- 최신 Python 문법으로 코드 변환
- 의존성 및 패키징 방식 현대화(`pyproject.toml` 등)
- 테스트 코드 개선 및 pytest로 포팅
- 자동화 및 CI/CD 적용

이 폴더에서 모든 업그레이드 실습을 진행하세요.
